🚀 Project Overview

In this post, I’ll walk you through how I designed, built, and deployed a full-stack productivity app called TaskMaster. The app helps users manage tasks, set priorities, and track their productivity daily.

🔧 Tech Stack Used

  • Frontend: HTML, CSS, JavaScript, React.js
  • Backend: Node.js, Express.js
  • Database: MongoDB
  • Deployment: Vercel (frontend), Render (backend)

🧠 Key Features

  • User registration and login
  • Task creation, editing, deleting
  • Priority tagging (high/medium/low)
  • Daily summary and productivity chart

Building this project helped me master API architecture, state management in React, MongoDB schema design, and CI/CD deployment flows. It also taught me the importance of UX — simplicity wins!
